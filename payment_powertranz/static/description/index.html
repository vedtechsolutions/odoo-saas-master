PowerTranz Payment Acquirer Integration

Accept Online Payments via PowerTranz

Integrate the PowerTranz payment gateway with your Odoo 18 instance to securely process online payments. This module supports credit card payments, 3D Secure authentication, tokenization, recurring payments, and more.

Key Features

  • Secure Payment Processing
  • 3D Secure 2.x Support
  • Tokenization (Save Cards)
  • Recurring Payments (Merchant & PowerTranz Managed)
  • Manual Capture (Authorize/Capture)
  • Full Refunds
  • Void Authorizations
  • Test Mode Support

Configuration

Configure the payment provider under Accounting or Website settings. Enter your PowerTranz credentials (ID, Password), enable desired features like 3DS and recurring payments, and set the operating mode (Test/Enabled).
