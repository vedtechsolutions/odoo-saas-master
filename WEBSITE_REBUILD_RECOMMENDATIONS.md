## Part 1: The Psychology of Why Customers Buy

### Understanding the Buyer's Brain

**The 3 Brain Model (Neuromarketing Foundation):**

| Brain Layer | Function | What It Wants |
|-------------|----------|---------------|
| **Reptilian ("Old Brain")** | Survival decisions | Safety, speed, simplicity |
| **Limbic (Emotional)** | Feelings, relationships | Trust, belonging, status |
| **Neocortex (Rational)** | Logic, analysis | Features, ROI, comparisons |

**Critical Insight:** The "old brain" makes buying decisions, then the rational brain justifies them. Your website must speak to ALL THREE, but lead with survival and emotion.

### The Real Reasons B2B Buyers Choose SaaS Solutions

**What they SAY they want:**
- "Best features"
- "Competitive pricing"
- "Good reviews"

**What they ACTUALLY want (deeper motivations):**

| Surface Desire | Deeper Motivation | Fear They're Avoiding |
|----------------|-------------------|----------------------|
| "Easy setup" | "I don't want to look incompetent" | Public failure, embarrassment |
| "Reliable hosting" | "I need to sleep at night" | Career-ending downtime |
| "Good support" | "Someone has my back" | Being left alone with problems |
| "Scalability" | "I'll succeed and need more" | Limiting my own growth |
| "Security" | "I won't be the breach headline" | Professional reputation destruction |

1. **"Is this safe?"** ‚Üí Established brand, security indicators

## Part 5: Trust Architecture

### Progressive Trust Building

**Stage 1: First Visit (0-30 seconds)**
- Professional design (visual credibility)
- Security badges visible
- No broken elements

**Stage 2: Exploration (30 seconds - 5 minutes)**
- Real customer testimonials with photos
- Specific numbers (not rounded)
- Clear contact information
- Company information accessible

**Stage 3: Consideration (5+ minutes)**
- Case studies with results
- Pricing transparency
- Clear refund/cancellation policies
- Technical documentation quality

**Stage 4: Decision (ready to buy)**
- Money-back guarantee prominent
- Easy cancellation policy stated
- Live chat available
- Phone number visible

# website_vedtech/controllers/pricing.py
from odoo import http
from odoo.http import request
import json

class PricingController(http.Controller):

    @http.route('/pricing', type='http', auth='public', website=True)
    def pricing(self):
        """Psychologically-optimized pricing page."""

        plans = request.env['saas.plan'].sudo().search([
            ('is_active', '=', True),
            ('is_trial', '=', False),
        ], order='monthly_price')

        # Find the "recommended" plan (middle tier for decoy effect)
        recommended_plan = None
        if len(plans) >= 2:
            recommended_plan = plans[len(plans) // 2]

        # Get social proof numbers
        instance_count = request.env['saas.instance'].sudo().search_count([])
        recent_signups = request.env['saas.subscription'].sudo().search_count([
            ('create_date', '>=', fields.Datetime.now() - timedelta(days=7))
        ])

        return request.render('website_vedtech.pricing_page', {
            'plans': plans,
            'recommended_plan': recommended_plan,
            'instance_count': instance_count,
            'recent_signups': recent_signups,
            'currency': request.env.company.currency_id,
        })

    @http.route('/pricing/calculate-savings', type='json', auth='public')
    def calculate_savings(self, plan_id, billing_cycle):
        """Show annual savings to trigger loss aversion."""
        plan = request.env['saas.plan'].sudo().browse(int(plan_id))
        if billing_cycle == 'yearly':
            monthly_total = plan.monthly_price * 12
            yearly_total = plan.yearly_price
            savings = monthly_total - yearly_total
            return {
                'savings': savings,
                'percentage': round((savings / monthly_total) * 100),
            }
        return {'savings': 0, 'percentage': 0}

## Part 9: Content Strategy

### Voice Guidelines

**Brand Personality:** Confident Expert Friend
- **Confident:** We know our stuff. No hedging language.
- **Expert:** Technical depth when needed, but accessible.
- **Friend:** Warm, helpful, never condescending.

**Avoid:**
- ‚ùå "We think..." (hedging)
- ‚ùå "World-class" (empty superlative)
- ‚ùå "Cutting-edge" (clich√©)
- ‚ùå "Simply the best" (unprovable)

**Use:**
- ‚úÖ Specific numbers ("3 minutes 42 seconds")
- ‚úÖ Customer quotes (real voices)
- ‚úÖ Outcome language ("Sleep through the night")
- ‚úÖ Honest limitations ("Not for teams under 3 users")

### SEO with Psychology

**Target Keywords by Buyer Stage:**

| Stage | Intent | Keywords |
|-------|--------|----------|
| Awareness | Learning | "odoo hosting options", "erp cloud solutions" |
| Consideration | Comparing | "odoo saas vs self-hosted", "best odoo hosting" |
| Decision | Buying | "odoo hosting pricing", "vedtech solutions reviews" |

**Title Tag Formula:**
`[Primary Keyword] | [Emotional Benefit] - VEDTECH Solutions`

Examples:
- "Odoo SaaS Hosting | Deploy in Minutes - VEDTECH Solutions"
- "Managed Odoo Hosting | Sleep Through the Night - VEDTECH"
